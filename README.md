# Sistema NOM-035-STPS-2018

Sistema web desarrollado en Laravel para la gestión y evaluación de factores de riesgo psicosocial conforme a la NOM-035-STPS-2018.

## Funcionalidades
- Gestión de usuarios (Administrador y Encargado)
- Aplicación de encuestas NOM-035
- Cálculo automático de resultados
- Historial de evaluaciones
- Panel administrativo

## Tecnologías
- PHP 8
- Laravel 10
- MySQL
- Bootstrap
- JavaScript

## Instalación
1. Clonar el repositorio
2. Ejecutar `composer install`
3. Configurar archivo `.env`
4. Ejecutar migraciones
